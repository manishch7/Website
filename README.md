# 🧠 Manish's Portfolio – Built with Vercel v0 (No-Code Prompting)

This is a personal portfolio website created entirely using [Vercel v0](https://v0.dev) — with **zero manual coding**, just powerful AI prompts. The source code was generated by AI, downloaded, and deployed — then pushed to GitHub.

> 🚀 Built with prompts, powered by Next.js, styled with Tailwind, and shipped with Vercel.

---

## 🔍 Project Overview

A clean, modern, and fully responsive portfolio built for showcasing projects, skills, and contact information. It was created with Vercel's [v0.dev](https://v0.dev), an AI UI builder that writes the code based on natural language prompts.

---

## 🧰 Tech Stack

| Technology      | Purpose                        |
|-----------------|--------------------------------|
| [Next.js](https://nextjs.org/)       | React Framework for building the site |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first styling framework       |
| [shadcn/ui](https://ui.shadcn.com/)  | Beautiful, accessible UI components   |
| [lucide-react](https://lucide.dev/)  | Icon library                          |
| [Vercel](https://vercel.com/)        | Hosting and deployment                |
